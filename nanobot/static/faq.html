General

OilGas Nanobot Swarm is a hierarchical multi-agent AI system built for petroleum engineering. It uses a 3-tier Queen ‚Üí L1 Domain Leads ‚Üí L2 Sub-Agents architecture to decompose complex engineering problems into specialized tasks and synthesize expert-level answers. What is the ECD formula used?
The standard industry formula:
ECD (ppg) = MW (ppg) + APL (psi) / (0.052 √ó TVD (ft))
Where MW = mud weight, APL = annular pressure loss, TVD = true vertical depth.
Which IPR model is used for production analysis?
Vogel (1968) for solution-gas drive wells:
q/q_max = 1 - 0.2(Pwf/Pr) - 0.8(Pwf/Pr)¬≤
Darcy radial flow for tight/conventional wells. The system generates a full 11-point IPR curve from a single test point.

Accuracy & Safety

How accurate are the calculations?
The engineering calculators use standard industry equations (Archie 1942, Vogel 1968, Hubbert & Willis 1957, Darcy-Weisbach, etc.) implemented in Python. Numerical accuracy is high for inputs within normal operating ranges.
‚ö†Ô∏è These are simplified models. They do not account for multiphase flow corrections, non-Darcy effects, or site-specific geological conditions. Always verify with full-physics simulators and licensed engineers before operational use.
Can I use this for actual well operations?
‚ö†Ô∏è No. This system is a decision-support tool for qualified petroleum engineers. Do not use it as a substitute for:
‚Ä¢ API-certified well control training or IADC WellSharp certification
‚Ä¢ Licensed engineer sign-off on well programs
‚Ä¢ Compliance with applicable BSEE/DOT/EPA regulations
‚Ä¢ Full-physics reservoir and drilling simulators
Use it for preliminary screening, education, and generating starting-point analyses that a licensed engineer then reviews and validates.
Does the AI ever make mistakes?
Yes. Large language models can hallucinate, misinterpret units, or apply incorrect formula variants. Always check: (1) the equation shown matches the standard reference, (2) inputs are in the correct units, (3) the result is in a physically reasonable range. The system is designed to show its work ‚Äî if a step looks wrong, it is wrong.

No. The default configuration uses Ollama Cloud and NVIDIA NIM as cloud AI backends ‚Äî no GPU, no local model downloads. If you want to run a local model with a GPU, set VLLM_URL to point to your vLLM server.
What API keys do I need?
Minimum required: one of
